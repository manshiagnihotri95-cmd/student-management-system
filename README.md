# Student Management System (GUI)

This is a GUI-based Student Management System built using Python and Tkinter.

## Features
- Add Student
- Show Students
- Delete Student
- Search Student

## Technologies Used
- Python
- Tkinter
- Object-Oriented Programming (OOP)

## Description
This project allows users to manage student records through a simple graphical interface. It helped me understand GUI development using Python.
